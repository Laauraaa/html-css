Tudo sobre fontes

Peso (negrito):

  • fonte mais cheia (negrito) ou fina
  • não serve para fontes simples (arial, verdana, times,...)
  • serve para fontes como Work Sans (import google)

    • font-weight:nome ou número;

    nome

    • lighter, número:100-500-900
    • normal, número:200-600
    • bold, número:300-700
    • bolder, número:400-800

    Estilo (itálico):

    font-style:valor; (noormal-italic)

    Decoração (sublinhado)

    text-decoration:underline;

    Shorthand font: simplificando o código das fontes

    Exemplo:

    • font-family:'Work Sans', sans-serif;
    • font-weight:bolder;
    • font-size:3em
    • font-style:italic;

    Simplificando:

    font:italic bolder 3em 'Work Sans', sans-serif;

    • A sequência sempre deve ser font-syle -> font-weight -> font-size -> font-family;
    • não precisa ter todas as propriedades, mas a sequência deve seguir.
    • Exemplo: font: bold 2em 'Work Sans';
